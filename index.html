British Airways APIs

  • This application is written using HTML5, CSS and Angular JS. As a first step, kindly register yourself for Developer APIs from their developer portal, https://developer.ba.com/
  • Ensure your browser is cross origin enabled. I used chrome browser. Create a shortcut and go to properties. Add this in target: > "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --user-data-dir="C:/Chrome dev session2" --disable-web-security
